Acessibilidade digital

Acessibilidade digital pode ser considerada como um conjunto de recursos e medidas que possibilitam a navegação, a compreensão e a interação de qualquer pessoa na internet, independentemente de condição e sem ajuda de ninguém.
Segundo o IBGE, o Brasil possui mais de 17 milhões de pessoas com deficiência. Isso sem considerar as que possuem outros tipos de limitação. É muita gente! Sendo assim, é preciso falar mais sobre acessibilidade digital. Acontece que, ao projetar um site, muitos ainda cometem o erro de associar acessibilidade apenas a ambientes físicos, esquecendo-se de que os meios digitais também precisam ser acessíveis.

Qual é a importância da acessibilidade digital

Para se ter uma noção da importância da acessibilidade digital, vejamos alguns números referente ao acesso à internet no Brasil: Segundo dados do Censo 2000, 80 milhões de brasileiros possuem acesso a internet, sendo que mais de 16 milhões são deficientes visuais. 31% navegam a uma velocidade lenta, de até 512 Kbps, revelou um estudo da The Nielsen Company, sobre o uso e qualidade das conexões de banda larga. Para acessar a internet, 40% da população brasileira utiliza smartphones, segundo estudo da IAB Brasil

Princípios básicos para um projeto web mais acessível

A W3C (Consórcio Internacional que desenvolve padrões para a Web), criou critérios que fazem parte do WCAG (Web Content Accessibility Guidelines) para orientar como os projetos web devem ser desenvolvidos, a fim de tornar o conteúdo mais acessível pelos usuários em geral. Entre os critérios apresentados estão, adicionar legenda em vídeos, incluir texto alternativo em imagens, usar cores adequadas para que o conteúdo apresentado seja legível. Para desenvolver um projeto web que ofereça acessibilidade, é preciso atender a quatro princípios básicos:

  • Percepção

    Os projetos devem permitir que os usuários sejam capazes de notar o conteúdo de que está sendo apresentado, sem qualquer dificuldade.

  • Funcionamento

    Ao desenvolver um projeto para web é preciso estudar como os usuários vão operar os componentes da interface bem como navegar no site, para que esta tarefa seja realizada de forma simples e intuitiva.

  • Compreensão

    É necessário compreender a informação que está sendo passada e conseguir operar a interface.

  • Robustez

    O projeto web deve ser acessível em antigas e novas tecnologias, de forma que os usuários entendam o que está sendo apresentado.
